Solomon dashboard

Tenant stats, API key, and outbound webhooks


Overview

Branding & embed theme

Drives /env.css and the chat widget. Theme default for embeds: auto follows the visitor’s OS; visitors can still override with the in-widget control.

Integration API key

Used for /api/integrations/v1/* inbound routes. Shown once after rotate.

Outbound webhooks

Signed POSTs with canonical schemaVersion / event. Empty event list = all events.

Endpoint Enabled Events

Auth: platform SSO cookie from /sso/callback. Cookie sessions require a CSRF token (fetched automatically before each change). For local dev you may use a Bearer token instead.

Developer

Storing Bearer tokens in the browser is an XSS risk and is discouraged for production / SOC 2; prefer SSO only.
